# World Cup Predictor

**Live app:** https://snpoudel-worldcup-predictor.streamlit.app

Predict World Cup 2026 knockout match scores with friends — from Round of 32 all the way to the Final.

## How to play

1. One person **creates a group** and shares the 6-character code
2. Friends **join with the code** — no login or signup needed
3. Everyone **enters their predicted score** for each match before it kicks off
4. Watch the **leaderboard** update as real results come in

## Scoring

| Prediction | Points |
|---|---|
| Correct outcome (win / draw / loss) | 1 |
| Exact scoreline | +1 bonus |

## Features

- No account required — just a name
- Shareable group code (or send the URL directly)
- Live scores sync automatically every hour
- Covers all 5 knockout rounds: R32 → R16 → QF → SF → Final
